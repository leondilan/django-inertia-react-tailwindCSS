# Django Inertia.js React Tailwind CSS

This is a simple example of how to use [Inertia.js](https://inertiajs.com/) with [React](https://reactjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the page by modifying `static/src/js/Pages/Home.jsx`.
